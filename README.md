# Azimuth Thruster
This repo contains parts and documentation for the azimuth thruster I created as part of my senior realization thesis at the University of Florida, "Design and Control of Azimuth Thrusters for Autonomous Surface Vehicles"

The goal of the project was to create a prototype of an azimuth thruster to be used on the Machine Intelligence Lab's (MIL) PropaGator 3 Autonomous Surface Vehicle (ASV). The project was succesful in creating a prototype for the thruster and demonstrating basic capabilities. The project also explored solving the thruster allocation problem with two fixed thrusters and two azimuth thrusters via a cost function.
